# Searching for Interpretable Multilayer Perceptrons Using a Structural--Behavioural Interpretability Metric
The proposed method is a hybrid optimisation framework for searching interpretable multilayer perceptrons (MLPs) for tabular classification problems. It combines Particle Swarm Optimisation and Genetic Algorithms with an interpretability-oriented objective tailored to one-hidden-layer networks. The objective integrates structural preferences (few active hidden neurons and a reduced set of input features) with a compact interpretability metric that penalises sign-inconsistent feature effects and dense input connectivity in the active part of the network. This metric guides model selection when predictive performance differences are not statistically significant, favouring simpler and more transparent MLPs. Experiments on a benchmark suite of datasets show that the interpretability-driven strategy yields substantially better interpretability scores and consistently reduces the number of active hidden neurons—thereby reducing the number of rules that are ultimately extracted from the MLP—while maintaining comparable test \textit{log-loss} to a parsimony-driven baseline. Overall, the results support the use of interpretability as a primary optimisation criterion for neural models within AutoML pipelines.
